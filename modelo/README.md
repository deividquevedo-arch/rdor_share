# Introdução
Repositório de padrão de estrutura para projetos de IA 


# Como utilizar
Atualmente o uso está limitado à máquina da pessoa, então você deve seguir os passos abaixo para subir a estrutura à um novo repositório. 
1.	conda create -y -n iatemplate python=3.12
2.  conda activate iatemplate
3.  pip install -r requirements.txt
4.  Clone o repositório destino na sua máquina e acesse-o
5.  Execute o comando a seguir e responda as perguntas cookiecutter ..\IATemplate\modelo\
6.  Envie o conteúdo para o git e comece seu trabalho pelo Databricks.
